# Resend and React Email with React Intl

This example shows how to use Resend and React Email with [React Intl](https://formatjs.github.io/docs/react-intl).

## Prerequisites

To get the most out of this guide, you’ll need to:

- [Create an API key](https://resend.com/api-keys)
- [Verify your domain](https://resend.com/domains)

## Instructions

1. Define environment variables in `.env` file:

  ```sh
cp .env.example .env
  ```

2. Install dependencies

  ```sh
npm install
  ```

3. Run Next.js app locally:

  ```sh
npm run dev
  ```

4. Open URL in your browser: 
  - http://localhost:3000/api/send/en
  - http://localhost:3000/api/send/pt
